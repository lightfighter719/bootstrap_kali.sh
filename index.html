Launch Kali Linux on macOS (Container)

Step 1: Create and start the Kali container

container run -it --name kali docker.io/kalilinux/kali-rolling bash

Step 2: Fix networking (if needed)

echo "nameserver 1.1.1.1" > /etc/resolv.conf
echo "nameserver 8.8.8.8" >> /etc/resolv.conf

Step 3: Update the system

apt update && apt full-upgrade -y
⚠️ Avoid installing system/kernel packages (like initramfs, dracut, systemd) inside containers. These can break the container environment.

Step 4: Install core Kali tools

apt install -y kali-tools-top10

Optional: Run bootstrap script (safer version)

curl -fsSL https://raw.githubusercontent.com/lightfighter719/bootstrap_kali.sh/main/bootstrap_kali.sh \
| sed -E '/initramfs|dracut|systemd/d' | bash

Step 5: Re-enter container later:

container exec -it kali bash
